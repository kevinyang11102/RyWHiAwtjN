## 内容介绍

本项目是一款基于Java和微信小程序的实验室教学日志系统，支持实验室内的教学日志记录、查询、统计等功能。通过本系统，教师和学生可以方便地记录和查看实验课程的教学过程，提高实验室教学的透明度和效率。此外，系统还提供了丰富的统计功能，便于管理员了解实验室教学的整体情况。
